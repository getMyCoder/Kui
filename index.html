介绍

介绍

Kui依赖于jQuery，插件和MVVM框架都是通过jQuery二次开发，主要目的是为了降低开发量！

栅格系统布局、按钮、以及背景的选取和Bootstrap相似

对于前端开发，因为大多数做的项目都有效果图，所以对于样式的内容较少，只是稍微写了点样式通用的样式

CSS

css样式中只是写了简单的样式，包括栅格系统、背景、字体颜色、边框颜色、圆角

插件

插件都是依赖于jQuery；一些较为常用的模态框、选项卡、折叠面板、瀑布流、banner等

MVVM

MVVM框架也是依赖于jQuery开发的，通过获取标签的属性进行判断、赋值、循环等（这里的MVVM框架只是做了数据的循环操作和事件操作）

文件下载地址https://github.com/getMyCoder/KuiZip

开始使用

引入css样式<link rel="stylesheet" type="text/css" href="Kui.css"/ >

因为依赖于jQuery，需要先将jQuery引入，然后引入<script type="text/javascript" src="Kui.js"></script>下面可以直接敲代码了！

栅格系统

栅格系统通过通过媒体查询，根据屏幕分辨率的大小进行合理的布局，一般分为手机(≤768)、平板(≥992)、PC(≥992)、大屏(≤1200),此处的大屏幕融合到了pc中

此处是按照12个格子进行分割的，里边的样式没有做任何的添加或是调整，只是做了布局处理！

手机≤768px 平板≥768px 桌面≥992px
分辨率 ≤768px ≥768px ≥992px
对应的类 kui-col-xs kui-col-sm kui-col-md
列数 12列

引用
<div class="kui-col-md1">1</div>
<div class="kui-col-sm1">1</div>
<div class="kui-col-xs1">1</div>

响应式布局
<div class="kui-col-md6 kui-col-sm4 kui-col-xs2">PC6、平板4、手机2</div>

注意：此处只是做了整体布局，里边的样式没有做处理，包括内部的左右浮动

弹窗

方法为:
Kui.open({
    title:"标题", //标题
    text:"内容", //这里是文本内容不能有标签
    message:"message", //这里可以有标签
    setSubmit:function () { //提交确定后执行的方法，如果不写则不执行任何方法
        console.log(1);
    }
});

执行方法的对象或是标签添加事件后执行这个方法

例如：<button class="kui-btn kui-primary" onclick=Kui.open()>点击我</button>

参数 值
title 弹窗的标题
text 弹窗的提示内容
message message和text不能同时存在，message里的内容可以用标签，设置样式
回调方法callback 提交确定后执行的方法，如果为空值不执行

Kui.promptBox({

执行方法的对象或是标签添加事件后执行这个方法

例如：
<button class="kui-btn kui-primary" onclick=Kui.promptBox({TipsMsg:'提交成功', TipsType:'danger'})>点击我</button>

参数 值
TipsMsg 提示信息
TipsType 提示框的背景颜色，包括success、danger、warning、info、primary

MVVM

MVVM介绍和使用

该框架依赖于jQuery库，主要是通过DOM更改、添加、循环数据！

标签:
<div id="main" class="main"></div>
获取这个标签的id传入方法中

方法：
Kui.KuiVM({
    id: "id",
    data: {},
    loads: function () {},
    method: {}
})

id:标签的id

data:请求的数据，或是静态数据

loads:这里是页面的DOm执行之前执行的内容，一般是ajax的请求;这是一个方法。里边直接定义方法的主体！

method:这里是DOM加载完毕后执行的事件，这是对象，里边的方法需要以对象的方法定义！

MVVM循环

注意：传入id的标签的一级子标签，最好不要做循环！！！

标签:
<div id="main" class="main">
<span m-for="msgs in arrs">{{msgs}}</span>
</div>

方法：
Kui.KuiVM({
    id: "id",
    data: {
        arrs: ["h", "e", "l", "l", "o", "!"],
    },
    loads: function () {},
    method: {}
})

循环的写法为：m-for,循环的对象为数组或是对象

对于循环对象中的某个数据的使用为：{{msgs.name}}、{{msgs.id}}、{{msgs.age}}

此处的循环仅仅是一级循环，对数组中的对象的数组循环是不支持的
例如：
json:{
    name:['张三','李四','王五'],
    name:['张三','李四','王五'],
    name:['张三','李四','王五']
}
不可以直接循环json中的name

数组排序

数组排序：

                方法:
                function sortArray(ArrayVal, flage) {
                    var sortArr = {
                        items: 0,
                        itemsVal: 0,
                        changeVal: 0,
                        newFlage: new Boolean()
                    };
                    for (var i = 0; i < ArrayVal.length; i++) {
                        sortArr.itemsVal = ArrayVal[i];
                        sortArr.items = i;
                        for (var j = i; j < ArrayVal.length; j++) {
                            sortArr.newFlage = (!flage) ? (sortArr.itemsVal > ArrayVal[j]) : (sortArr.itemsVal < ArrayVal[j]);
                            if (sortArr.newFlage) {
                                sortArr.itemsVal = ArrayVal[j];
                                sortArr.items = j;
                            }
                        }
                        sortArr.changeVal = ArrayVal[i];
                        ArrayVal[i] = sortArr.itemsVal;
                        ArrayVal[sortArr.items] = sortArr.changeVal;
                    }
                    return ArrayVal;
                }

该方法会把数组排序后的数组返回出来！

参数 值
ArrayVal 传入的数组
flage Boolean值，false为正序，true为倒序，默认为false

数据中的json排序

数组中的json按照id或是其他的参数进行排序：

 方法:
 function sortArrayJson(ArrayJsonVal, Indexes, flage) {
    var sortArr = {
        items: 0,
        itemsVal: 0,
        changeVal: 0,
        itemsJson: null,
        newFlage: new Boolean()
    };
    for (var i = 0; i < ArrayJsonVal.length; i++) {
        sortArr.itemsVal = ArrayJsonVal[i][Indexes];
        sortArr.items = i;
        for (var j = i; j < ArrayJsonVal.length; j++) {
            sortArr.newFlage = (!flage) ? (sortArr.itemsVal > ArrayJsonVal[j][Indexes]) : (sortArr.itemsVal < ArrayJsonVal[j][Indexes]);
            if (sortArr.newFlage) {
                sortArr.itemsVal = ArrayJsonVal[j][Indexes];
                sortArr.items = j;
                sortArr.itemsJson = ArrayJsonVal[j];
            }
        }
        sortArr.changeVal = ArrayJsonVal[i];
        ArrayJsonVal[i] = sortArr.itemsJson;
        ArrayJsonVal[sortArr.items] = sortArr.changeVal;
    }
    return ArrayJsonVal;
 }

该方法排序的对象是数组中的元素全部为json，按照json中的元素Indexes进行排序

参数 值
ArrayJsonVal 传入的数组
Indexes 按照数组元素为json的，json中的某一个元素进行排序
flage Boolean值，false为正序，true为倒序，默认为false 
                    例如：
                    var Arr = [
                        {id: 7, name: 'asdf7'},

                        {id: 5, name: 'asdf5'},
                        {id: 2, name: 'asdf2'},
                        {id: 3, name: 'asdf3'},
                        {id: 6, name: 'asdf6'},
                        {id: 0, name: 'asdf0'},
                        {id: 22, name: 'asdf22'},
                        {id: 88, name: 'asdf88'},
                        {id: 4, name: 'asdf4'}
                    ];

以上数组按照id进行排序
sortArrayJson(Arr,"id")

数组添加、删除

数组排序：

                方法:
                 function AddRemoveArray(ArrayVal, AddArray, flage) {
                    if (!flage) {
                        ArrayVal.push(AddArray);
                        return ArrayVal
                    } else {
                        if (ArrayVal.indexOf(AddArray) != -1) {
                            ArrayVal.splice(ArrayVal.indexOf(AddArray), 1);
                            return ArrayVal
                        }
                    }
                }

该方法是对数组的元素进行添加或是删除操作！

参数 值
ArrayVal 传入的数组
AddArray 传入的元素
flage Boolean值，false添加，为把AddArray元素添加进数组，true删除，把AddArray元素从数组中删掉，如果数组中不存在AddArray，则不进行任何操作！
